HR Analytics – Employee Attrition Analysis

This project analyzes employee attrition using HR data to identify key factors contributing to employee turnover. The objective is to support HR teams by providing data-driven insights that can help improve retention strategies and workforce planning.

Key Insights:
• Attrition trends across departments and experience levels
• Factors influencing employee turnover
• Visual analysis for easier interpretation

Tools Used:
Python, Pandas, Matplotlib, Seaborn
